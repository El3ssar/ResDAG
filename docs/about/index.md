# About

ResDAG is PyTorch-native reservoir computing, written by Daniel Estevez-Moya
and released under the MIT license. The code lives at
[github.com/El3ssar/resdag](https://github.com/El3ssar/resdag); everything
meta about the project lives on the four pages below.
