# Get started

This section covers installation, the training/forecasting workflow, and two
worked examples. Read [Installation](installation.md) first if the package is not
yet on your system.

**[Installation](installation.md)** — `pip` / `uv`, optional extras, CUDA notes.

**[Mental model](mental-model.md)** — reservoir layer, readout fitting, warmup vs
train vs forecast, and the role of `f_warmup`.

**[Your first ESN](your-first-esn.md)** — sine series with `prepare_esn_data` and
`classic_esn`.

**[Lorenz walkthrough](lorenz-walkthrough.md)** — chaotic 3D series with `ott_esn`
and the same data utilities.

For task-oriented workflows (HPO, ensembles, file I/O), see
[Guides](../guides/index.md). For signatures and types, see
[Reference](../reference/index.md).
