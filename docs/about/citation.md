# Citation

If you use ResDAG in academic work, please cite the software and the underlying
reservoir-computing literature relevant to your method (ESN, NG-RC, etc.).

## Software

```bibtex
@software{resdag2024,
  author = {Estevez-Moya, Daniel},
  title = {ResDAG: PyTorch-native reservoir computing},
  url = {https://github.com/El3ssar/resdag},
  version = {0.4.0},
  year = {2024},
}
```
