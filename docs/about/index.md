# About ResDAG

**ResDAG** is a PyTorch-native library for Echo State Networks, Next-Generation
Reservoir Computers, and graph-structured recurrent initialization.

- **Author:** Daniel Estevez-Moya
- **License:** MIT
- **Repository:** [github.com/El3ssar/resdag](https://github.com/El3ssar/resdag)
- **PyPI:** [pypi.org/project/resdag](https://pypi.org/project/resdag/)
