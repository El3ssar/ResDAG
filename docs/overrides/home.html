Ten lines to a trained forecaster

One teacher-forced pass collects reservoir states; one conjugate-gradient ridge solve fits the readout. Training a 500-neuron forecaster takes well under a second — model selection over thousands of candidates is a for-loop, not a cluster job. 

import resdag as rd

data = rd.utils.load_file("lorenz.npy")          # (batch, time, 3)
warmup, train, target, f_warmup, val = rd.utils.prepare_esn_data(
    data, warmup_steps=200, train_steps=5000, val_steps=2000)

model = rd.models.ott_esn(reservoir_size=500, feedback_size=3, output_size=3)
rd.ESNTrainer(model).fit((warmup,), (train,), targets={"output": target})

prediction = model.forecast(f_warmup, horizon=2000)   # autoregressive
